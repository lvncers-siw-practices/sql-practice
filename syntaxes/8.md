# Basic Syntax 8

## 追加

```sql
INSERT INTO
    student
    (id, name, birthday)
VALUES
    (6, "小西　真奈美", "1977-10-27")
;
```

```sh
Query OK, 1 row affected (0.016 sec)

mysql> select * from student;
+----+--------------------+------------+-------+
| id | name               | birthday   | class |
+----+--------------------+------------+-------+
|  1 | 佐藤 琢磨          | 1977-01-28 | CG    |
|  2 | 大塚 愛            | 1982-09-09 | Web   |
|  3 | 藤井 隆            | 1972-03-10 | Web   |
|  4 | 福原 愛            | 1988-11-01 | CG    |
|  5 | 大黒 将志          | 1980-05-04 | NULL  |
|  6 | 小西　真奈美       | 1977-10-27 | NULL  |
+----+--------------------+------------+-------+
6 rows in set (0.000 sec)
```

再度実行すると下のエラーになる。
主キーには重複した値を入れられない。

```sh
ERROR 1062 (23000): Duplicate entry '6' for key 'student.PRIMARY'
```

## 更新

```sql
UPDATE
    student
SET
    class = "Web"
WHERE
    id = 6
;
```

## 削除

```sql
DELETE FROM
    student
WHERE
    id = 6
;
```
