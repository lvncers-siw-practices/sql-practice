# SQL Syntaxes 3

## 合計

```sql
SELECT SUM(score) AS 合計点 FROM exam WHERE id = 1;
```

```sh
+-----------+
| 合計点     |
+-----------+
|       169 |
+-----------+
1 row in set (0.006 sec)
```

## 平均

```sql
SELECT AVG(score) AS 平均点 FROM exam WHERE id = 1;
```

```sh
+-----------+
| 平均点     |
+-----------+
|   84.5000 |
+-----------+
1 row in set (0.005 sec)
```

## 最大

```sql
SELECT MAX(score) AS 最低点 FROM exam;
```

```sh
+-----------+

|        92 |
+-----------+
1 row in set (0.001 sec)
```

## 最小

```sql
SELECT MIN(score) AS 最低点 FROM exam;
```

```sh

+-----------+
1 row in set (0.002 sec)
```

## 件数

```sql
SELECT COUNT(*) AS 件数 FROM exam WHERE score >= 80;
```

```sh
+--------+
| 件数   |
+--------+
|      3 |
+--------+
1 row in set (0.003 sec)
```

### 注意：NULL はカウントされない

カラムないにデータが入っているかを明示的に確認する時に使用する。

```sh
mysql> SELECT COUNT(class) AS 件数 FROM student;
+--------+
| 件数   |
+--------+
|      4 |
+--------+
1 row in set (0.003 sec)
```

```sh
mysql> SELECT COUNT(ALL class) AS 件数 FROM student;
+--------+
| 件数   |
+--------+
|      4 |
+--------+
1 row in set (0.003 sec)
```

```sh
mysql> SELECT COUNT(DISTINCT class) AS 件数 FROM student;
+--------+
| 件数   |
+--------+
|      4 |
+--------+
1 row in set (0.003 sec)
```
