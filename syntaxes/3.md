# SQL Basic Syntaxes

## 合計

```sql
SELECT SUM(score) AS 合計点 FROM exam WHERE id = 1;
```

```sh
+-----------+
| 合計点     |
+-----------+
|       169 |
+-----------+
1 row in set (0.006 sec)
```

## 平均

```sql
SELECT AVG(score) AS 平均点 FROM exam WHERE id = 1;
```

```sh
+-----------+
| 平均点     |
+-----------+
|   84.5000 |
+-----------+
1 row in set (0.005 sec)
```
