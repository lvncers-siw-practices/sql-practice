# SQL Basic Syntaxes

## 実行順序

```sql
SELECT
   ~
FROM
   ~
WHERE
   (条件を指定)
GROUP BY
   (グループ化)
HAVING
   (グループ化後の条件を指定)
ORDER BY
   (ソート)
(LIMIT)
```

## dmpファイルインストール

```sh
mysql -u root -p school < school.dmp
```

## エラーの対処法

syntax error → まずはタイポを疑う。特にエラー文で指定している文の直前の単語あたりを疑う。

### 並び替え

指定をしないと並び順は不定となる。
そのため、複数件が該当する場合には、必ず並び替えを指定すること。

#### 昇順（デフォルトで省略可）

```sql
SELECT * FROM exam ORDER BY score ASC;
```

#### 降順

```sql
SELECT * FROM exam ORDER BY score DESC;
```
