## 全科目受験者の exam 行を取る話

```sql
SELECT e.*
FROM exam AS e
WHERE e.id IN (
  SELECT id
  FROM exam
  GROUP BY id
  HAVING COUNT(DISTINCT subject) = (
    SELECT COUNT(DISTINCT subject) FROM exam
  )
);
```

```sql
SELECT s.name, e.subject, e.score
FROM student AS s
JOIN exam AS e
  ON s.id = e.id
WHERE s.id IN (
  SELECT id
  FROM exam
  GROUP BY id
  HAVING COUNT(DISTINCT subject) = (
    SELECT COUNT(DISTINCT subject) FROM exam
  )
);
```

```sql
SELECT * FROM exam WHERE id != 3 GROUP BY subject;
```

ERROR 1055 (42000): Expression #1 of SELECT list is not in GROUP BY clause and contains nonaggregated column 'school.exam.id' which is not functionally dependent on columns in GROUP BY clause; this is incompatible with sql_mode=only_full_group_by

ONLY_FULL_GROUP_BY 違反って呼ばれてる。
エラー文にそのまま書いてあるやつ👇

`sql_mode=only_full_group_by`

意味としては、

GROUP BY に含まれていない列を、集計関数もつけずに SELECT に書いちゃダメ

```sql
SELECT subject, AVG(score) FROM exam WHERE id != 3 GROUP BY subject;
```

```SQL
SELECT subject, AVG(score) FROM exam WHERE id = 3 GROUP BY id;
```

```sql
SELECT id, SUM(score) "合計点", AVG(score) "平均点" FROM exam WHERE id != 3 GROUP BY id;
```

## 南極を除いた大陸ごとの平均寿命で７０歳以上のものを大きい順に

```sql
SELECT
    continent,
    ROUND(AVG(LifeExpectancy), 1) AS avg_lifeexpectancy
FROM
    country
WHERE
    Continent != "Antarctica"
GROUP BY
    Continent
HAVING
    avg_lifeexpectancy >= 70
ORDER BY
    avg_lifeexpectancy DESC
;
```
