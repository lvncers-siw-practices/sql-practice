### 6. 地域(Region)がポリネシア、メラネシア、ミクロネシアで、面積が500㎢以上、独立した年(IndepYear)が分かる国

```SQL
SELECT
    Name,
    Region,
    SurfaceArea,
    Indepyear
FROM
    country
WHERE
    Region IN ("polynesia", "melanesia", "micronesia") AND
    SurfaceArea >= 500 AND
    Indepyear IS NOT NULL
ORDER BY
    Region,
    Name;
```

第二ソートキーの考え方としては同じRegionがあった時に次にソートする条件は何かを考えること。

`AND` は `OR` より先に評価される。
つまり、`Region` の条件文で `IN` を使わないで `OR` で繋ぐと別の結果になる。

## GROUP 演習問題

### 4. 10カ国以上の国がある大陸と国の数を国の多い順に表示

```SQL
SELECT
    Continent,
    COUNT(*) AS country_cnt
FROM
    country
GROUP BY
    continent
HAVING
    country_cnt >= 10
ORDER BY
    country_cnt DESC,
    continent ASC;
```

### 5. 国ごとに使われている言語数が１０以上の国コードと言語数を言語数の多い順に表示

```SQL
SELECT
    CountryCode,
    COUNT(*) AS lang_num
FROM
    countrylanguage
GROUP BY
    CountryCode
HAVING
    lang_num >= 10
ORDER BY
    lang_num DESC,
    CountryCode ASC
;
```
