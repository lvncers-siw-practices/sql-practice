### 1. 人口の少ない都市ランキングの下位5都市

```SQL
SELECT Name, Population FROM City ORDER BY Population ASC LIMIT 5;
```

### 2. 平均寿命が40歳未満の国

```SQL
SELECT name,LIfeexpectancy FROM country WHERE lifeexpectancy < 40 ORDER BY lifeexpectancy desc;
```

### 3. Unitedで始まる国の名前

```SQL
SELECT
    code,
    name
FROM
    country
WHERE
    name LIKE 'United%'
ORDER BY
    name ASC;
```

### 4. 首都がない国の数

```SQL
SELECT
    COUNT(*) AS No_Capital_Count
FROM
    country
WHERE
    Capital IS NULL;
```

### 5. 地域(Region)がカリブ諸島(Caribbean)の平均GNP, 平均GNPold

```SQL
SELECT
    ROUND(AVG(GNP),1) as "GNP_AVG",
    ROUND(AVG(GNPOld),1) as "GNPOld_AVG"
FROM
    country
WHERE
    region = "Caribbean";
```

### 6. 地域(Region)がポリネシア、メラネシア、ミクロネシアで、面積が500㎢以上、独立した年(IndepYear)が分かる国

```SQL
SELECT
    Name,region,Surfacearea,Indepyear
FROM
    country
WHERE
    region in ("polynesia", "melanesia", "micronesia") and
    surfacearea >= 500 and
    indepyear is not null
ORDER BY
    region, indepyear;
```

## GROUP 演習問題

### 4. 10カ国以上の国がある大陸と国の数を国の多い順に表示

```SQL
SELECT
    Continent,
    COUNT(*) AS country_cnt
FROM
    country
GROUP BY
    continent
HAVING
    country_cnt >= 10
ORDER BY
    country_cnt DESC,
    continent ASC;
```

### 5. 国ごとに使われている言語数が１０以上の国コードと言語数を言語数の多い順に表示

```SQL
SELECT
    CountryCode,
    COUNT(*) AS lang_num
FROM
    countrylanguage
GROUP BY
    CountryCode
HAVING
    lang_num >= 10
ORDER BY
    lang_num DESC,
    CountryCode ASC
;
```
