## 7. 世界の都市の数

同じ名前の都市名だが、国が違う都市がいくつか存在し、`DISTINCT` を使うことは意味が変わってしまう。

```sql
SELECT
    count(*)
FROM
    city;
```

## 8. 世界の言語の数

```sql
SELECT
    COUNT(DISTINCT language) AS LangNum
FROM
    countrylanguage;
```

## 9. 日本の首都圏の人口

```sql
SELECT
    SUM(Population) AS CapitalAreaPopulation
FROM
    city
WHERE
    District IN (
        'Tokyo-to',
        'Kanagawa',
        'Chiba'
    );
```

## 10. malanesia 地域の人口

```sql
SELECT
   concat('国名:', region, '人口', population, '人')
FROM
   country
WHERE
   region = 'melanesia'
ORDER BY
   region,
   name;
```
